## Getting started

### Tools needed on your machine

* Git
* Node

### Initial configuration

```bash
> git clone https://github.com/FIN-18-20/RubiksMind.git
> cd RubiksMind/client
> npm i
> npm run dev
```

### Test the setup

Try to reach:
* The Vue SPA on [localhost:8080](http://localhost:8080)
